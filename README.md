# ChatSystem

## Characteristics
- Written in TypeScript
- Uses Node.js and the NestJS framework

## Differential
It includes a frontend part, which will be made available in the future, also written in TypeScript but using Next.js.

## Dependencies

[Nest](https://github.com/nestjs/nest)
[Node.js](https://github.com/nodejs/node)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
